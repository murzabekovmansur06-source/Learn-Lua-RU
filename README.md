# 🇷🇺 Изучаем Lua — с нуля до игр в Roblox

Бесплатный курс Lua на русском языке. Для тех кто никогда не программировал.

## 📚 Уроки

| Урок | Тема | Что узнаешь |
|------|------|-------------|
| 1 | Переменные | Числа, строки, вывод на экран |
| 2 | Условия | if, else, elseif |
| 3 | Циклы | for, while, повторение |
| 4 | Функции | Создание и вызов |
| 5 | Таблицы | Списки и словари |
| 6 | Roblox | Первый скрипт в игре |

## 🎯 Для кого

- Никогда не программировал
- Не знаешь английский
- Хочешь делать игры в Roblox

## 🚀 Как начать

1. Начни с урока 1
2. Читай объяснение в README.md
3. Запускай код из .lua файла
4. Меняй код и экспериментируй

## 📝 Формат

Каждый урок содержит:
- **README.md** — объяснение на русском
- **.lua файл** — готовый код с комментариями
