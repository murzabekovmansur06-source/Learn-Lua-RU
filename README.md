
# 🇷🇺 Изучаем Lua — с нуля до игр в Roblox

Бесплатный курс Lua на русском языке. Для тех кто никогда не программировал.

## 📚 Уроки

| Урок | Тема | Что узнаешь |
|------|------|-------------|
| 1 | Переменные | Числа, строки, вывод на экран, типы данных |
| 2 | Условия | if, else, elseif, and, or, not |
| 3 | Циклы | for, while, вложенные циклы, break |
| 4 | Функции | Создание, аргументы, return |
| 5 | Таблицы | Списки, словари, pairs, ipairs |
| 6 | Roblox | Первый скрипт, создание объектов |
| 7 | События | Touched, Click, PlayerAdded |
| 8 | RemoteEvent | Клиент и сервер, FireServer, OnServerEvent |
| 9 | DataStore | Сохранение прогресса игрока |
| 10 | GUI | Кнопки, текст, меню, интерфейс |

## 🎯 Для кого

- Никогда не программировал
- Не знаешь английский
- Хочешь делать игры в Roblox

## 🚀 Как начать

1. Начни с урока 1
2. Читай объяснение
3. Копируй код в Roblox Studio
4. Меняй код и экспериментируй
5. Делай задания в конце каждого урока

## 📝 Формат

Каждый урок содержит:
- Подробное объяснение на русском
- Готовый код с комментариями
- Задания для практики
- Частые ошибки и как их избежать
